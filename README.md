# Tic Tac Toe website

> Đây là dự án thiết kế trang web trò chơi Tic Tac Toe ở phía Frontend.

### Công nghệ sử dụng:

- ReactJS.

### Chức năng:

- Hiển thị bảng trò chơi Tic Tac Toe có 9 ô.

- Cho phép người chơi thay phiên nhau bằng cách nhấp vào ô trên bảng.

- Có nút reset game.

- Lưu trạng thái của trò chơi vào localStorage.
